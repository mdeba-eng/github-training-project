# 🚀 مشروع تدريب الموبايل الشخصي (github-training-project)

### 📝 وصف المشروع
تطبيق موبايل تعريفي شخصي بسيط تم بناؤه باستخدام Flutter. يحتوي التطبيق على شريط علوي (AppBar)، كرت عرض معلومات شخصية (Profile Card)، وتذييل سفلي (Footer). تم بناء المشروع لتطبيق مهارات Git & GitHub في الكلية الجامعية للعلوم التطبيقية (UCAS).

---

### 🛠️ التقنيات المستخدمة (Technologies)
* **Flutter & Dart:** لبناء واجهات المستخدم المتجاوبة للهواتف الذكية.
