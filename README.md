# 🚀 مشروع تدريب الويب الشخصي (github-training-project)

### 📝 وصف المشروع
موقع تعريفي شخصي بسيط يحتوي على شريط تنقل (Navbar)، قسم معلومات شخصية، وتذييل (Footer). تم بناء المشروع لتطبيق المهارات الأساسية والمتقدمة في إدارة المشاريع البرمجية والتعاون باستخدام Git & GitHub في الكلية الجامعية للعلوم التطبيقية (UCAS).

---

### 🛠️ التقنيات المستخدمة (Technologies)
* **HTML5:** لبناء الهيكل الأساسي للصفحة والـ Components المطلوبة.
* **CSS3:** لتنسيق المظهر وتصميم كرت معلومات احترافي متجاوب (Responsive).
